# Hypr-vogix

> **Archived** — This project has been integrated into [vogix](https://github.com/i-am-logger/vogix) as **vogix shader**. All future development continues there. This repository is no longer maintained.

---

Hypr-vogix was a monochromatic screen overlay for Hyprland. It generated GLSL shaders and applied them via Hyprland's native `decoration:screen_shader` to create immersive monochromatic experiences.
